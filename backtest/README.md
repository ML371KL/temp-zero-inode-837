# Бэктест «Разлом-26»

Репликация скоринга панели на исторических данных 2006–2026 с честными лагами
публикации каждого ряда. Использован для аудита v4.8 (июль-2026).

- `engine.py` — загрузка данных, построение баллов всех индикаторов, семейная
  агрегация, композит, гейт, детекторы; варианты улучшений V1–V8.
- `run.py` — метрики стратегии против buy&hold, IC, эпизоды, распределения.
- `run2.py` — правило восстановления, чувствительность порогов, эпизоды PROTECT.

Данные: `../histdata/*.csv` (FRED через API, workflow histdata.yml) + Yahoo
(^GSPC, ^SP500TR, BTC-USD, GC=F — качаются локально) + полная история HY/IG OAS
собрана из публичных зеркал FRED-выгрузок (сверены с API-окном бит-в-бит:
максимальное расхождение 0.0).

Ключевые результаты: см. абзац «Валидация бэктестом» в методологии на странице.
